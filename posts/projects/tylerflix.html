TylerFlix

Self-Hosted Media
Docker + Traefik Jellyfin Hardware Transcoding

My self-hosted streaming stack with custom branding, hardened ingress, and automated backups. Friends and family get a polished landing page; I get full control and observability.

Highlights

  • GPU-accelerated transcoding for smooth remote playback.
  • Jellyseerr handles requests; Uptime Kuma monitors every service.
  • Nightly object storage backups and pruning scripts to stay lean.

Stack

  • Docker Compose with Traefik for TLS + routing.
  • Cloudflare DNS + Argo tunnels for secure exposure.
  • Custom dashboard reusing this site’s glassmorphism UI.

Outcome

  • Reliable home media hub with zero cloud vendor lock-in.
  • Simple onboarding: one link for guests, automated certs.
  • Low-maintenance: alerts, backups, and cleanups on autopilot.
